# WAN Video Generator — Mock Service (API + UI)

Этот проект — минимальный, но полностью рабочий пример сервиса, который:

- принимает изображение,
- отправляет его в модель (пока заглушка),
- генерирует видео на основе входного изображения,
- возвращает это видео пользователю,
- отображает результат через Gradio UI.

Структура уже подготовлена для интеграции настоящей модели (например, WAN).

```bash
project/
├── api/ # FastAPI backend (генерация видео)
│ ├── main.py # Основной HTTP-сервис
│ ├── model_service.py # Логика инференса модели (сейчас заглушка)
│ ├── video_*.mp4 # Видео-заглушка, имитирующее вывод модели
│ ├── outputs/ # Папка, куда сохраняются сгенерированные ролики
│ ├── requirements.txt # Python зависимости API
│ └── Dockerfile # Docker-образ для backend
│
├── ui/ # Gradio клиентский интерфейс
│ ├── ui.py # Web UI: загрузка картинки + отображение видео
│ ├── requirements.txt # Python зависимости UI
│ └── Dockerfile # Docker-образ для UI
│
├── docker-compose.yml # Общий запуск API + UI
└── README.md # Документация
```

1. Пользователь загружает изображение через UI.
2. UI отправляет картинку в API endpoint /inference_by_image.
3. API передаёт байты изображения в ModelService.
4. Заглушка копирует заранее подготовленный video_2025-12-11_*.mp4.
5. API возвращает mp4-файл как результат.
6. UI отображает видео пользователю через встроенный плеер.
